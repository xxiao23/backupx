## Welcome to BackupX

BackupX allows you to manage backups for your MySQL and Postgres DBs easily.

### Key features

- Create backup schedules as a part of the DB schema.
- Query backup status using INFORMATION_SCHEMA.
- Restore a backup to a live DB.
